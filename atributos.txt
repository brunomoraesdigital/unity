==================================================
SISTEMA BASE DO PERSONAGEM
==================================================

NÍVEL MÁXIMO
NV máximo = 100
(personagem inicia no nível 1)

PONTOS DE ATRIBUTO
+1 ponto por nível ganho
nível 1 não possui pontos para distribuir
total máximo distribuível = 99 pontos

Todos os atributos começam em 1.

--------------------------------------------------
ATRIBUTOS PRIMÁRIOS
--------------------------------------------------

FOR  = força
AGI  = agilidade
VIT  = vitalidade
INT  = inteligência
DES  = destreza
SOR  = sorte
NV   = nível

--------------------------------------------------
EXP
--------------------------------------------------

EXP necessária para subir de nível:
EXP_para_subir =
10 * (NV ^ 1.5)

EXP recebida ao derrotar inimigos:
EXP_ganha =
EXP_base_monstro * (NV_monstro / NV_jogador)

Regras:
- inimigos de nível menor dão menos EXP
- inimigos de nível maior dão mais EXP
- evita farm infinito em monstros fracos

--------------------------------------------------
ATRIBUTOS SECUNDÁRIOS
--------------------------------------------------

ATAQUE MÁGICO
atk_mag_min = (INT * 2) + (DES / 2) + SOR
atk_mag_max = (INT * 4) + (DES / 2) + SOR

ATAQUE FÍSICO (corpo a corpo)
atk_fis_min = (FOR * 1) + (DES / 2) + SOR
atk_fis_max = (FOR * 3) + (DES / 2) + SOR

ATAQUE FÍSICO (longa distância)
atk_dist_min = (DES * 1) + (AGI / 2) + SOR
atk_dist_max = (DES * 3) + (AGI / 2) + SOR

--------------------------------------------------
COMBATE
--------------------------------------------------

VELOCIDADE DE ATAQUE
vel_ataque = (AGI * 2) + DES

ESQUIVA
esquiva = (AGI * 2) + (DES / 2) + SOR

PRECISÃO
precisao = (DES * 2) + (AGI / 2) + SOR

--------------------------------------------------
CRÍTICO E ESQUIVA PERFEITA
--------------------------------------------------

acerto_critico =
floor(SOR / 2) * floor(DES / 10)

esquiva_perfeita =
floor(SOR / 2) * floor(AGI / 10)

Regras:
- mínimo considerado = 1%
- máximo considerado = 99%
- multiplicadores nunca podem ser menores que 1

--------------------------------------------------
VIDA E MANA
--------------------------------------------------

vida_max =
(VIT * 3 + NV * 2) * 100

mana_max =
(INT * 3 + NV * 2) * 100

--------------------------------------------------
REGENERAÇÃO
--------------------------------------------------

taxa_regen_percentual =
0.5 + (NV * 0.03)

rec_vida =
vida_max * taxa_regen_percentual

rec_mana =
mana_max * taxa_regen_percentual

--------------------------------------------------
INTERVALO DE REGENERAÇÃO
--------------------------------------------------

tempo_regen_vida_seg =
10 / (1 + (VIT / 50))

tempo_regen_mana_seg =
10 / (1 + (INT / 50))

Menor valor = regeneração mais rápida
(valor representa segundos entre cada recuperação)

--------------------------------------------------
DEFESA
--------------------------------------------------

defesa =
((VIT * 3) + (FOR / 2) + (NV * 2)) * 100

Defesa reduz dano por valor fixo (não percentual).

--------------------------------------------------
CAPACIDADE DE CARGA
--------------------------------------------------

carga_max =
((FOR * 10) + (NV * 2)) * 100
